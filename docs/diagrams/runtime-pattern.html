Agent Runtime — Template Method Pattern

BaseAgentRuntime defines the lifecycle; runtimes override hooks

Abstract BaseAgentRuntime
final execute(profile, text, options) Orchestrates: build args → spawn → write stdin → capture output → parse result
abstract getCommand() CLI binary name (e.g. "claude", "opencode")
abstract buildExecuteArgs(profile) Runtime-specific CLI flags and arguments
virtual writeInput(child, profile, text) Write prompt to stdin (overridable — Codex uses CLI arg)
virtual formatInput(text, context) Format text for PTY write (shared context injection)
virtual buildExecuteEnv(profile, env) Runtime-specific environment variables
